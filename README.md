# skin-cancer-diagnosis-using-transcriptome-sequence-
Skin cancer is a major global issue, with millions of cases yearly. This study proposes an LLM-based AI framework using RNA-Seq data to detect early biomarkers. Combining BioBERT, Geneformer, W2V-Att, ETEO, and DENN, the model improves accuracy (96.78%) and enhances early diagnosis with better feature selection and reduced redundancy.
